# ALCANCE TÉCNICO – FASE 3

## Arquitectura Implementada

Modelo cliente-servidor basado en Node.js y Express.

### Componentes:

- Backend API REST
- Controladores modulares
- Estructuras en memoria para evidencia técnica
- Registro de eventos (bitácora)

---

## Endpoints Implementados

### Salud del sistema
GET /health

### Usuarios
POST /users

### Documentos
POST /documents
GET /documents

### Bitácora
GET /logs

---

## Funcionalidades Implementadas

- Creación de usuarios con rol.
- Registro de documentos con metadatos.
- Generación automática de eventos de auditoría.
- Consulta de bitácora consolidada.
- Versionado básico de documentos.

---

## Evidencia Técnica

- Repositorio versionado.
- Modularización por controladores.
- Separación de responsabilidades.
- API funcional mínima ejecutable.

---

## Resultado

Se cuenta con un sistema mínimo funcional que cumple con los requerimientos detectados en Fase 2 y permite trazabilidad verificable.
