# ACTA DE INICIO – FASE 2

Proyecto: PKF Digital Security Platform  
Fase: Levantamiento de Assets para Desarrollo de Software de Seguridad  
Periodo estimado: Enero 2025 – Mayo 2025  

## Objetivo

Realizar inventario detallado de activos digitales, infraestructura, usuarios y flujos documentales para definir los requerimientos técnicos del sistema de seguridad y auditoría.

## Alcance

- Identificación de servidores y repositorios
- Inventario de bases de datos
- Clasificación de tipos documentales
- Identificación de usuarios y roles actuales
- Análisis preliminar de riesgos
- Definición de criticidad por activo

## Resultado Esperado

Contar con matriz formal de activos y riesgos que sirva como base técnica para el desarrollo del sistema en Fase 3.
